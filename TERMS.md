# GAM Back Office — UI Terminology · Нэр томьёоны жагсаалт

Every user-facing term, label, button, column header, navigation item, status,
placeholder, tooltip and message in the app, extracted from `index.html` and
`assets/app.js` (`assets/styles.css` contains no user text apart from a 🔍 search icon).

**Notes**
- Where the app itself shows a bilingual label (`Mongolian · English`), both columns hold the app's exact wording.
- Where a string appears in only one language in the code (most toast/validation messages and some headers are Mongolian-only), the other column is a faithful translation for reference.
- `₮` = MNT (Mongolian tögrög). Date format throughout: `YYYY-MM-DD`.
- Sorted by Location: sidebar → topbar → gate → dashboard → investors → statement → subscribe → redeem → transactions → funds → nav → bonds → instruments → fees → ecif → common.
