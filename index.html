Тавтай морил

GAM бэк-офис системийн өгөгдөл нь таны OneDrive дээрх нэг Excel файлд хадгалагдана. Эхлэхийн тулд өгөгдлийн файлаа сонгоно уу.

All data is stored in a single Excel workbook on your OneDrive. Pick or create it to begin.

Заавар · How it works
  1. OneDrive хавтас доторх GAM_BackOffice.xlsx файлыг сонгоно (эсвэл шинээр үүсгэнэ).
  2. Та өгөгдөл оруулах бүрд "Хадгалах" дарна — OneDrive автоматаар үүлэн рүү синк хийнэ.
  3. Дараагийн удаа апп нь сүүлд нээсэн файлыг санаж, шууд холбогдоно.

Microsoft Edge эсвэл Google Chrome ашиглана уу (File System Access API шаардлагатай).
