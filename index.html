Тавтай морил

GAM бэк-офис системд нэвтрэхийн тулд Голомт Ассет Менежментийн Microsoft 365 акаунтаар нэвтэрнэ үү. Өгөгдөл таны OneDrive дээрх GAM_BackOffice.xlsx файлд хадгалагдана.

Sign in with your Golomt Asset Management Microsoft 365 account. Data is stored in GAM_BackOffice.xlsx on your OneDrive — created automatically if it doesn't exist.

Заавар · How it works
  1. «Microsoft-ээр нэвтрэх» дарж GAM-ийн Microsoft 365 акаунтаар нэвтэрнэ.
  2. Систем таны OneDrive дээрх GAM_BackOffice.xlsx файлыг автоматаар олно — байхгүй бол шинээр үүсгэнэ.
  3. «Хадгалах» дарах бүрд файл OneDrive рүү шинэчлэгдэнэ.
  4. Дараагийн удаа ижил акаунтаар нэвтрэхэд өгөгдөл автоматаар ачаалагдана.

Microsoft Edge эсвэл Google Chrome ашиглана уу.
